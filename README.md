# WanderLog 🌍

A React-based travel bucket list application that allows users to explore countries, view detailed country information, and maintain a personal wishlist and visited countries list.

## Tech Stack

- React (JSX)
- Vite
- CSS
- React Router DOM
- Axios
- REST Countries API
- Reqres API

## Features

- User Login and Signup
- Protected Routes
- Session Persistence using Local Storage
- Country Search Functionality
- Region-based Country Filtering
- Country Detail Page
- Travel Bucket List
- Visited Countries Tracking
- Responsive Desktop Layout
- Loading and Error Handling

## APIs Used

### REST Countries API

- Get all countries
- Get country details by country code

### Reqres API

- User Login
- User Registration

## How to Run Locally

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open the application in your browser

```text
http://localhost:5173
```
